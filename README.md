To run the project You will need some reasonable IDE (preferrably IntelliJ Idea). There are restrictions about using github api regarding number of requests per hour, therefore to use the code You will need github api Token and insert it into application.yml. API has one endpoint to receive basic information about users repositories, that are also not forked.
